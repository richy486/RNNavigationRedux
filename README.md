## Instalation

```shell
brew install node
brew install watchman
npm install -g react-native-cli
```

## Running

run with

```shell
react-native run-ioss
```

or 

```shell
react-native run-android
```
(You must have an android emulator running or a device connected)

## Logging

```shell
react-native log-ios
```
